就绪

模型管理

配置大模型 API Key，在右侧聊天窗口中选择使用。

向量检索配置

配置 embedding 服务后，AI 可对你的文件做语义搜索（按意思找，不只按关键词）。 注：文件内容会发送到该服务计算向量。

CLI 管理

CLI 工具和外部智能体。内置条目可查看不可删除，手动添加的可以编辑和删除。

MCP 服务器

配置 MCP（Model Context Protocol）CLI 服务器，为 AI 扩展外部工具能力。启用后 AI 可在对话中直接调用这些工具。

本地文件 内置
AI 可读写你绑定的文件夹：列出目录、读取/写入文件、全文搜索。右键侧边栏文件夹 →「绑定 AI 路径」启用。
list_dir read_file write_file search_files

系统角色

选择一个角色，AI 将按照该角色的专业身份思考和回复。相当于给 AI 预设一个「专家」人格。

智能体 = 角色提示词 + 模型 + 工具白名单。创建不同智能体来处理不同类型的任务。

将智能体关联到具体项目，聊天时自动使用该项目的默认智能体。

用量统计

查看自身 AI 调用用量统计，或监控本机其他 AI 工具的运行状态。

操作日志

记录每次 Agent 调用：用了什么模型、调了什么工具、产出了什么文件、耗时多少。

数据监测

新媒体数据聚合查询。需先配置数据源 API Key。
