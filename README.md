# Seegent — 项目管理指挥部

> **你管项目，AI 管执行。**  
> Seegent 是你每天第一件事打开的地方——看数据、管待办、分配任务、验收产出。
> 写代码/做图/批量自动化这些重活，交给 WorkBuddy、Cursor 等外部智能体。

```text
每天早上：打开 Seegent → 看项目看板 → 管待办 → 把任务分配给外部 AI → 验收产出物
```

---

## Seegent 做什么 vs 不做什么

| Seegent 做（指挥官） | 外部 AI 做（兵） |
|-----|-----|
| 📊 看项目数据，跟踪指标 | 💻 写代码 |
| 📋 整理待办，排优先级 | 🎨 做图、设计 |
| 📁 管理文件，看产出物 | ⚡ 批量自动化 |
| 🤖 让 AI 帮你分析数据、总结进展、生成周报 | 📝 写长文档 |
| 🔌 连接外部数据源（飞书等） | 🔧 复杂调试 |

---

## 核心功能

| 模块 | 一句话说明 |
|------|-----------|
| 🏠 **首页看板** | 每日工作入口：各项目数据、待办、最近活动，一眼看清 |
| 📁 **文件管理** | 侧边栏文件树 + 多标签编辑器，支持拖拽、多格式预览 |
| 💬 **AI 对话** | 多会话、流式输出、多模型切换，AI 帮你管项目 |
| 🔍 **智能搜索** | 全文检索 + 语义搜索，"我去年写的定价笔记"秒找 |
| 🔌 **数据接入** | 飞书多维表格等外部数据源，自动同步指标 |
| 🔧 **模型管理** | DeepSeek、OpenAI、Claude、智谱等，自定义 Provider |

---

## 架构

```
浏览器 (File System API + IndexedDB)  ←→  Python HTTP Server (Agent + SQLite)

                      ↕ 读写同一文件夹

          外部智能体（WorkBuddy / Cursor / Claude Code）
```

---

## 快速开始

```bash
pip install seegent
seegent
```

浏览器访问 `http://localhost:8765`。
